# To Do List / Tasker
## React + javascript

### tecnologias e conteudo utilizados
- React
    - javascript
    - local storage
- create-react-app

---
### pré requisitos: 
 - node.js
 
### iniciando aplicação com create-react-app
- instalando dependencias
    ```console
    npm install
    npm install @types/react @types/react-dom
    ```
- rodando aplicação
    ```console
    npm start
    ```
